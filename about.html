ABOUT THIS PROJECT

This site is a hands-on sandbox for learning the basics of the web: HTML for structure, CSS for styling, and simple multi-page navigation.

Every page you see here is a plain text file written by hand -- no page builder, no template engine. That's intentional: seeing exactly how a line of code turns into a line on screen is the fastest way to actually understand it.

The eventual goal is to take what's learned here and apply it to a real business website.
